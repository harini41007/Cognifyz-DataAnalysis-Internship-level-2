# Cognifyz Data Analysis Internship - Level 2

## Project Overview
This project focuses on analyzing restaurant data to uncover valuable insights related to customer ratings, cuisine preferences, geographical distribution, and restaurant chain performance.

## Key Analysis
### Customer Rating Analysis
+ Analyzed rating distribution
+ calculated  the average number of customer votes
+ Identified the most common rating values

### Cuisine Trend Analysis
+ Identified popular cuisins combinations
+ Explored popular cuisine combination
+ Examined customer preferences across cuisines

### Geographic Analysis
+ Visualized restaurant locations using interactive maps.
+ Applied clustering techniques to identify restaurant concentration areas.

### Restaurant Chain Analysis
+ Identified restaurant chains with multiple outlets
+ Compared chain based on ratings and customer votes
+ Evaluated the popularity and performance of restaurant brands.

### Tools and Libraries
+ Python
+ pandas
+ NumPy
+ Matplotlib
+ seaborn
+ Folium
+ Scikit-learn
 ### Outcome
 The project demonstrates how data analysis, visualization, geospatial mapping,and machine learning techniques can be used to derive actionable insights from restaurant datasets and understand customer behavior more effectively
